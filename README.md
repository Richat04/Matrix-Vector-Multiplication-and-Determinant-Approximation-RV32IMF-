# Matrix-Vector-Multiplication-and-Determinant-Approximation-RV32IMF-
RISC-V assembly program that performs:   1. Matrix–Vector Multiplication: Yi = Σ A[i][j] × X[j] for a 6×6 floating-point matrix A  and 6×1 vector X.   2. Determinant Computation det(A) by triangularization and gaussian method   3. Comparison Check: Compare ΣY and det(A)
